# Harbor — только iOS

Мобильное приложение для **iPhone (iOS)**: анонимные письма между незнакомцами.  
PWA (добавить на экран «Домой») — без App Store.

## Запуск на iPhone

1. Раздайте проект по HTTPS (нужно для PWA и ES modules), например через [ngrok](https://ngrok.com/) или свой сервер.
2. Откройте ссылку в **Safari** на iPhone.
3. Нажмите **Поделиться** → **На экран «Домой»**.
4. Запускайте приложение с иконки — откроется полноэкранно, без адресной строки.

## Локальный тест (симуляция на компе)

Чтобы проверить верстку без телефона, запускайте **наш сервер** (он отдаёт `index.html` для путей вроде `/home`, `/read/letter-1` — иначе будет 404):

```bash
cd /Users/noashapiro/Downloads/harbor-app
python3 serve.py
```

Откройте в браузере: **http://localhost:8765/**  
Обновление страницы на любом маршруте (/home, /profile и т.д.) будет работать.  
Включите режим эмуляции iPhone (DevTools → Toggle device toolbar), чтобы увидеть safe area и размеры под iOS.

## Что настроено под iOS

- **viewport-fit=cover** и safe area (вырез, Dynamic Island, низ экрана).
- **Режим приложения:** полноэкран, чёрный полупрозрачный статус-бар.
- **Портрет:** ориентация только portrait.
- **Касания:** без подсветки тапов, плавная прокрутка, удобные зоны нажатия (≥44pt).
- **manifest.json** для «Добавить на экран «Домой»».

## Иконки

Положите в `assets/`:

- **icon-180.png** (180×180) — иконка на главном экране iPhone.
- При необходимости: **icon-192.png**, **icon-512.png** для манифеста.

## Структура проекта

- **index.html** — оболочка приложения, только под iOS.
- **styles/** — tokens, base, components, screens (все с учётом safe area и touch).
- **js/** — app, router, state, mock-data, экраны и компоненты.
- **manifest.json** — PWA для iOS.
- **harbor-ui.html** — старый одностраничный прототип (только для справки).

Приложение рассчитано **только на iOS**; другие платформы не поддерживаются.
