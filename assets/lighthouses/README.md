# Изображения маяков для аватара

**Сюда загрузите картинки маяков** — одно из них будет случайно выбрано как аватар пользователя при нажатии «Set sail» (создание аккаунта).

## Имена файлов

Положите в эту папку файлы с именами:

- `1.jpg`, `2.jpg`, `3.jpg`, … до `20.jpg`

или в формате PNG:

- `1.png`, `2.png`, … до `20.png`

Рекомендуется: **квадратные или вертикальные изображения**, минимум **200×200 px**.  
Приложение случайно выберет один из загруженных файлов (по номерам 1–20) при создании аккаунта.

## Пример

```
assets/lighthouses/
  1.jpg
  2.jpg
  3.png
  ...
  20.jpg
```

Если файла с выбранным номером нет, покажется стандартный аватар (иконка маяка).
